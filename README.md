# Лабораторная работа по Питону №4

---
## Цель
Реализовать систему управления библиотекой

---

## Реализованные пункты:

- класс ```Book```
- списковая коллекция ```BookCollection```
- пользовательская коллекция ```IndexDict```
- классы библиотек ```BaseLibrary``` -> ```Library``` -> ```DigitalLibrary```(цифровая библиотека)
- Псевдослучайная симуляция с пунктами:
    - ```add_book``` - добавление книги
    - ```remove_book``` - удаление книги
    - ```search_books``` - поиск книги
    - ```update_index``` - обновление индекса
    - ```borrow_book``` - выдача читателю
    - ```return_book``` - возврат читателем
    - ```check_nonexistent``` - существование книги в библиотеке
 ---

 ## Запуск 
 ```python -m src.main.py```

 ## Вывод 

 В ходе данной лабораторной работы были освоены навыки ООП в Python, созданы пользовательские коллекции с необходимым функционалом, а также создана реализация псевдослучайности.
