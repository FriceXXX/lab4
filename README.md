# Лабораторная работа по Питону №4

---
## Цель
- Реализовать систему управления библиотекой
- Освоить работу с псевдослучайностью
- Освоить реализацию пользовательских моделей

---

## Реализованные пункты:

- класс ```Book``` - класс книги с аттрибутами ```title```, ```author```, ```year```, ```genre```, ```isbn```.
- списковая коллекция ```BookCollection``` - коллекция для создания и манипуляции с коллекциями книг (доступна функция фильтрации поо году, автору и жанру) и поддержкой срезов.
- словарная пользовательская коллекция ```IndexDict``` - коллекция для создания и обновления индексов книг, а также поиска книг по индексам.
- классы библиотек ```BaseLibrary```(основные функции - добавление/удаление книг) -> ```Library```(переопределение основных функций, добавление функций для выдачи/возврата книг и поиска -> ```DigitalLibrary```(цифровая библиотека: переопределение функций выдачи/возврата в связи с добавлением поля ```digital_copies```) - наследование.
- Псевдослучайная симуляция с пунктами и поддержкой ```seed```:
    - ```add_book``` - добавление книги
    - ```remove_book``` - удаление книги
    - ```search_books``` - поиск книги
    - ```update_index``` - обновление индекса
    - ```borrow_book``` - выдача читателю
    - ```return_book``` - возврат читателем
    - ```check_nonexistent``` - существование книги в библиотеке
- все магические методы(```__contains__```, ```__repr__```, ```__call__```, ```__add__```)
- все коллекции поддерживают итерирование(```__iter__```), длину(```__len__```) и доступ по индексу(```__getitem__```).
